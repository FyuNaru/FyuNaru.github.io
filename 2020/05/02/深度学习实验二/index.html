实验要求

  • 基于PyTorch实现AlexNet [1]结构
  • 在Cifar-10数据集上进行验证
  • 使用tensorboard进行训练数据可视化 Loss曲线
  • 如有条件，尝试不同参数的影响，尝试其他网络结构
  • 请勿使用torchvision.models.AlexNet

环境配置

实验环境如下：

  • Win10
  • python3.8
  • Anaconda3
  • Cuda10.2 + cudnn v7
  • GPU : NVIDIA GeForce MX250

另外，为实现可视化，需配置安装tensorboard的环境，执行以下命令即可实现：

python –m pip install tensorboardX

conda install tensorflow

tensorboard的运行方法如下：

打开pycharm的terminal，输入tensorboard --logdir result

然后在浏览器打开localhost:6006即可.

具体的可视化操作参考的是这篇博客:
Pytorch使用tensorboardX可视化。超详细

参数调整

本实验不同于实验一，需要调整的参数较多，经过一系列的调参，现对调参过程做一个经验总结：

首先，由于原先的AlexNet输入的图片大小为224 * 224，而本实验使用的为32 * 32，大小相差很多，因此卷积核需要调小，在这里我五层卷积层使用的卷积核大小均为3 * 3。

其次，卷积核的数量也需要进行一定比例的减小，这里我使用的依次为48，96，192，192，96。一开始使用的卷积核的数量较少，结果不是很好，在将卷积核的数量调多后，结果有了一定的改善。

stride和padding使用的统一为1。

全连接网络中的节点数量，一开始设置的较大，后来调整到512时，可以较快的将准确率收敛到75%

batch_size设置为128，当batch_size较小时，可能出现结果很差的情况，这里我依次尝试了16,32,64,128，然后从中选取了结果较好的128

优化器的lr参数为0.1
