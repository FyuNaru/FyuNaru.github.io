输入输出和模板

  1. 与输入输出流操作相关的类：istream, ostream, ifstream, ofstream, iostream, fstream
  2. 标准流对象：cin, cout, cerr, clog
  3. 输出重定向
    freopen("test.txt", "w", stdout);//将标准输出重定向到test.txt文件
    freopen("test.txt", "r", stdin);//将标准输入重定向到test.txt文件
  4. 判断输入流结束
    对于cin >> x，当输入为文件时，读到文件末尾时，此式的值为false；当输入为键盘时，按“ctrl+z”输入结束。
